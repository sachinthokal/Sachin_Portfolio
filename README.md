## 🚀 Getting Started

Follow these steps to run the project on your local machine:

```bash
# 1. Clone the repository
git clone https://github.com/sachinthokal/sachin-portfolio.git

# 2. Navigate to the project folder
cd sachin-portfolio

# 3. Install dependencies
npm install

# 4. Start the project
npm start

# 5. Open in browser (if not automatically opened)
# Usually accessible at http://localhost:3000
```
